sling-contenteditor
===================

Apache Sling content descriptor editor for eclipse. 
This plugin helps to edit sling content.json files. It is based on [cnd-editor](https://github.com/evra/cnd-editor) 
which provides CND definitions editing features.

Main features
--
* Text based content editing
* Syntax highlighting
* CND Autocompletion
* Validation based on CND definitions
* Simple reafactoring
